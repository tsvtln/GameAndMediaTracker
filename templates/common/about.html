About CheckPoint

This website is a project for SoftUni Django Advanced.

CheckPoint is designed to help you keep track of your game progress, download and share save files and access ROMs for your favorite games. The navigation bar and homepage showcase the main features such as
ROMs downloading and uploading;
Save Files (SAVE VAULT) downloading, uploading;
Community Forums, Events and much more.

You can find the source code on GitHub:
https://github.com/tsvtln/GameAndMediaTracker
